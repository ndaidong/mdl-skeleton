# Usage

```
git clone https://github.com/ndaidong/mdl-skeleton.git
cd mdl-skeleton

// install and run with Node.js
npm install

// load external resource if any
npm run setup

// start server
npm start

```

If everything goes well, you can get the output at: http://localhost:8081
